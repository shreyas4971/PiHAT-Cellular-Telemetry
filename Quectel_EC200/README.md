Scripts tailored for the PiHAT EC200, focusing on high-speed Type-C data transfer and its dedicated onboard GNSS hardware.

**1. ec200_uart_basic.py:** Description: Establishes UART serial communication with the Quectel EC200 via Raspberry Pi GPIO pins. Ideal for low-bandwidth AT command testing, basic network registration, and lightweight MQTT publishing without tying up USB ports.

**2. ec200_typec_master.py:** Description: The primary telemetry script utilizing the high-speed USB Type-C interface. Features complete 4G MQTT session management alongside highly accurate, hardware-driven GNSS (GPS/GLONASS) satellite polling.

**3. ec200_rtc_setup.py:** Description: Initializes and polls the onboard DS3231 Real-Time Clock over the I2C bus. Ensures payloads generated by the Quectel module maintain precise, localized timestamping even during total network outages.
